#Spring-Init<br />
项目说明:
本项目是一个基于Spring一站式架构快速开发的初始化项目.<br />
Spring各个XML配置简单化.技术基本选择最新.<br />
包结构尽量一目了然.<br />
本项目尚利用业余时间在孵化中.<br />
